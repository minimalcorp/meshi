# 使い方

## 1. 食品マスタを登録する

「食品」画面で、よく食べるものを食品マスタとして登録します。1食（1単位）あたりの基準 kcal を入れておくと、以降の記録が選択するだけで済みます。

- 名前（例: ごはん）
- 1食あたりの kcal（例: 250）
- 単位ラベル（任意。例: 「1杯」「100g」）

不要になった食品はアーカイブできます（記録済みのデータは保持されます）。

## 2. 摂取カロリーを記録する

「記録」画面から、食品マスタを選んで摂取記録を追加します。

- 数量を掛けて実値の kcal を記録可能
- 朝・昼・晩などの食事区分を指定
- 複数の食品を同じ食事としてまとめて記録（バスケット方式）

## 3. 目標を設定する

「目標」画面で 1 日あたりの目標摂取カロリーを登録します。目標は期間ごとに変更でき、変更後も過去の記録は当時の目標で評価されます。

## 4. 進捗を可視化する

「ダッシュボード」で、目標に対する進捗とカロリー推移を確認できます。

- 日次・週次・月次の集計
- 目標に対する達成度
- カロリー推移のグラフ（visx）
